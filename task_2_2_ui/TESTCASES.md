# TESTCASES.md

## Общая информация
**Проект**: Платформа модерации объявлений
**Цель**: Покрыть все пользовательские сценарии из задания 2.2
**Окружение**: 
- Десктоп: viewport >= 1024px
- Мобильная версия: viewport <= 768px, isMobile: true

## 1. Десктопная версия (Desktop)

### TC-01. Главная страница - Фильтр "Диапазон цен"
**Предусловия**: Открыта главная страница (/)

**Шаги и проверки**:

**1.1 Позитивный сценарий (валидный диапазон)**
1. В поле "Цена от" ввести 1000
2. В поле "Цена до" ввести 5000
3. Применить фильтр
**Ожидаемый результат**: В списке только объявления с ценой от 1000 до 5000 включительно.

**1.2 Негативный сценарий - Ввод некорректных символов**
1. В поле "Цена от" ввести abc!@#
2. В поле "Цена до" ввести 5000
**Ожидаемый результат**: Поле игнорирует некорректные символы / очищается / показывает ошибка.

**1.3 Негативный сценарий - Отрицательные значения**
1. В поле "Цена от" ввести -500
2. В поле "Цена до" ввести 3000
**Ожидаемый результат**: Значение приводится к 0 или показывается ошибка.

**1.4 Негативный сценарий - "От" больше "До"**
1. В поле "Цена от" ввести 5000
2. В поле "Цена до" ввести 1000
**Ожидаемый результат**: Ошибка валидации или значения меняются местами.

### TC-02. Главная страница - Сортировка "По цене"
**Предусловия**: Открыта главная страница (/)

**Шаги**:
1. Выбрать "Сортировать по" - "Цене"
2. Выбрать "Порядок" - "По возрастанию"
3. Дождаться обновления списка
4. Выбрать "Порядок" - "По убыванию"

**Ожидаемый результат**:
- По возрастанию: цены от меньшей к большей
- По убыванию: цены от большей к меньшей

**Проверки**:
- Цены в первых 5-7 карточках идут в правильном порядке

---

### TC-03. Главная страница - Фильтр "Категория"
**Предусловия**: Открыта главная страница (/)

**Шаги**:
1. Выбрать категорию (например "Электроника")
2. Дождаться обновления списка
3. Сбросить фильтр (выбрать "Все категории")

**Ожидаемый результат**:
- После выбора категории - только объявления этой категории
- После сброса - все категории

**Проверки**:
- Все видимые карточки имеют выбранную категорию

---

### TC-04. Главная страница - Тоггл "Только срочные"
**Предусловия**: Открыта главная страница (/)

**Шаги**:
1. Включить тоггл "Только срочные"
2. Дождаться обновления списка
3. Выключить тоггл

**Ожидаемый результат**:
- При включённом тоггле - только срочные объявления с бейджем
- При выключенном - все объявления

---

### TC-05. Страница Статистики - Управление таймером
**Предусловия**: Открыта страница /stats

**Шаги**:
1. Нажать кнопку "Обновить"
2. Нажать кнопку "Отключить автообновление"
3. Подождать 5 секунд
4. Нажать кнопку "Включить автообновление"
5. Подождать 5 секунд

**Ожидаемый результат**:
- "Обновить" - данные обновились
- "Остановить" - данные не меняются 5 секунд
- "Запустить" - данные автоматически обновляются

## 2. Мобильная версия (Mobile)

### TC-06. Переключение между темной и светлой темой
**Предусловия**: Viewport установлен 375x667, открыта любая страница

**Шаги**:
1. Кликнуть на переключатель темы (светлая - темная)
2. Кликнуть повторно (темная - светлая)

**Ожидаемый результат**:
- Тема переключается в обе стороны
- Изменения применяются ко всему интерфейсу

**Проверки**:
- У html атрибут data-theme меняется (light - dark)
- После перезагрузки тема сохраняется
