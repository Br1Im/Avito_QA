## UI Дефекты

### Критические (P0)

1. **Геофильтр не работает**
   - Краткое описание: При выставленной геолокации "Москва" в выдаче появляется объявление "Персидский котёнок" из Новосибирска
   - Шаги воспроизведения:
     1. Открыть /list
     2. Установить геолокацию "Москва"
     3. Применить фильтр
   - Фактический результат: В результатах поиска отображаются объявления из других городов
   - Ожидаемый результат: Отображаются только объявления из выбранного региона
   - Серьёзность: P0 — критический
   - Окружение: Desktop, Chrome

2. **Фильтр по цене игнорируется**
   - Краткое описание: При установленном диапазоне цен 1000-30000 в выдаче присутствуют объявления за 500 и 40000
   - Шаги воспроизведения:
     1. Открыть /list
     2. Установить диапазон цен 1000-30000
     3. Нажать применить
   - Фактический результат: Отображаются объявления с ценами 500 и 40000
   - Ожидаемый результат: Отображаются только объявления с ценой в диапазоне 1000-30000
   - Серьёзность: P0 — критический
   - Окружение: Desktop, Chrome

3. **Плюшевый кот среди живых**
   - Краткое описание: В разделе поиска котят отображается объявление с плюшевой игрушкой
   - Шаги воспроизведения:
     1. Открыть /list
     2. Выбрать категорию "Котята"
     3. Просмотреть результаты
   - Фактический результат: Среди живых котят отображается плюшевая игрушка
   - Ожидаемый результат: Отображаются только живые котята
   - Серьёзность: P1 — высокий
   - Окружение: Desktop, Chrome

## Причины дефектов (предположения)

- **Фильтры (цена, гео, пол, порода)** — вероятно проблема на backend (игнорирование параметров запроса)
- **Плюшевый кот** — ошибка в категоризации или тегах объявления
- **Ошибка под header** — проблема с API или fallback-логикой
- **UI-баги (сетка)** — проблемы в верстке, grid или адаптив

## Дополнительные проверки, рекомендуемые к выполнению

- Проверка комбинаций фильтров (гео + цена + порода + пол)
- Проверка граничных значений цены (0, отрицательные, максимальные)
- Проверка работы фильтров в других категориях
- Проверка других фильтров (возраст, цвет)
- Нагрузочное тестирование API
- Проверка безопасности (CORS, SQL injection, XSS)
