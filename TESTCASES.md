# TESTCASES.md — Тест-кейсы API сервиса объявлений

## Описание API

| № | Метод | Endpoint | Описание |
|---|-------|----------|----------|
| 1 | POST | `/api/1/item` | Создать объявление |
| 2 | GET | `/api/1/item/:id` | Получить объявление по ID |
| 3 | GET | `/api/1/:sellerID/item` | Получить все объявления продавца |
| 4 | GET | `/api/1/statistic/:id` | Получить статистику по объявлению |
| 5 | DELETE | `/api/2/item/:id` | Удалить объявление по ID |
| 6 | GET | `/api/2/statistic/:id` | Получить статистику v2 |

### Тело запроса POST /api/1/item

```json
{
  "sellerID": 123456,
  "name": "string",
  "price": 1000,
  "statistics": {
    "likes": 0,
    "viewCount": 0,
    "contacts": 0
  }
}
```

### Ответ

```json
{
  "id": "string",
  "sellerId": 123456,
  "name": "string",
  "price": 1000,
  "statistics": {
    "likes": 0,
    "viewCount": 0,
    "contacts": 0
  },
  "createdAt": "string"
}
```
