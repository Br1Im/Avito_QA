# Тест-кейсы

## API эндпоинты

| Метод | Endpoint | Что делает |
|-------|----------|------------|
| POST | `/api/1/item` | Создать объявление |
| GET | `/api/1/item/:id` | Получить по ID |
| GET | `/api/1/:sellerID/item` | Все объявления продавца |
| GET | `/api/1/statistic/:id` | Статистика объявления |
| DELETE | `/api/2/item/:id` | Удалить объявление |
| GET | `/api/2/statistic/:id` | Статистика v2 |

Тело POST-запроса:
```json
{
  "sellerID": 123456,
  "name": "string",
  "price": 1000,
  "statistics": { "likes": 0, "viewCount": 0, "contacts": 0 }
}
```

## API тесты (Задание 2.1)

### Создание объявления (POST /api/1/item)

**Позитивные:**

| ID | Что проверяю | Техника | Ожидание |
|----|--------------|---------|----------|
| TC-001 | Валидные данные — всё заполнено корректно | Эквивалентное разделение | 200, id возвращён, поля сохранены |
| TC-002 | Цена = 0 (нижняя граница) | Граничные значения | 200 или 400 с ошибкой |
| TC-003 | Цена = 999999999 (верхняя граница) | Граничные значения | 200 |
| TC-004 | Статистика нулями | Классы эквивалентности | 200, инициализация нулями |
| TC-005 | Статистика ненулевая (likes:10, viewCount:50) | Эквивалентное разделение | 200, данные сохранены |
| TC-006 | Несколько объявлений одного продавца → GET список | Комбинаторика | Все возвращаются |
| TC-007 | Разные sellerID → разные id | Классы эквивалентности | Уникальные id |
| TC-008 | Кириллица в имени («Персидский котёнок») | Именное разделение | 200, имя сохранилось |

**Негативные:**

| ID | Что проверяю | Техника | Ожидание |
|----|--------------|---------|----------|
| TC-010 | Нет sellerID | Классы эквивалентности | 400 |
| TC-011 | Нет name | Классы эквивалентности | 400 |
| TC-012 | Нет price | Классы эквивалентности | 400 |
| TC-013 | Пустое тело `{}` | Классы эквивалентности | 400 |
| TC-014 | Цена отрицательная (-100) | Негативные значения | 400 |
| TC-015 | Статистика отрицательная (likes:-5) | Негативные значения | 400 |
| TC-016 | sellerID = 0 (вне диапазона) | Граничные значения | 400 |
| TC-017 | sellerID строкой `"111111"` | Несоответствие типов | 400 или конвертация |
| TC-018 | price строкой `"1000"` | Несоответствие типов | 400 или конвертация |
| TC-019 | Content-Type: text/plain, не JSON | Классы эквивалентности | 400 |
| TC-020 | name = пустая строка | Граничные значения | 400 |

| TC-050 | POST на GET-endpoint | Неправильный метод | 405 |

**Корнер-кейсы:**

| TC-027 | price = null | Граничные значения | 400 |
| TC-030 | Эмодзи в имени | Unicode | 200 |
| TC-031 | Иероглифы в имени | Unicode | 200 |
